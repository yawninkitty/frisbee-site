manage.py - файл для проведения операция создания файлов, миграций, запуска локального сервера

frisbee - корневая папка, содержащая весь проект

asgi и wsgi - нужны для выгрузки сайта на сервер

settings.py
BASE_DIR - полный путь к проекту
SECRET_KEY - секретный ключ приложения для безопасности
перед выгрузкой проекта на сервер секретный ключ нужно поменять
DEBUG - показ ошибок на страницах сайта
при выгрузке на сервер убрать на False
ALLOWED_HOSTS - хосты для публикации, домены ссылками
INSTALLED_APPS - приложения сайта
MIDDLEWARE - поддержка, классы, плагины внутри проекта
ROOT_URLCONF - url ссылки сайта
DATABASES - база данных

url.py - отслеживание url-адресов

Запуск сервера: python manage.py runserver
Остановить сервер: ctrl+C
